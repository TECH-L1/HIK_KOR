HikCentral Professional 개요

+

HikCentral Professional은 비디오, 출입 통제, 보안 관리 및 기타 다양한 시스템을 통합하여 중앙에서 관리할 수 있도록 지원하는 통합 보안 관리 플랫폼입니다.

핵심 기능

🔧 장치 관리 (Device Management)

다양한 보안 장치들의 통합 관리 및 실시간 모니터링

👥 사용자 & 차량 관리

사용자 권한 설정 및 차량 출입 관리 시스템

🏢 구역 관리 (Zone Management)

건물 및 구역별 세분화된 보안 정책 적용

🚪 출입 통제 (Access Control)

출입 권한 관리 및 출퇴근 시스템 연동

🚨 경보 모니터링 (Alarm Management)

실시간 이벤트 감지 및 즉시 알림 시스템

🤖 스마트 분석 (AI Analytics)

AI 기반 영상 분석 및 행동 패턴 인식

System Requirements

+

  • Windows 11 (64비트)
  • Windows 10 (64비트)
  • Windows Server 2025, 2022, 2019, 2016
  • Windows Server 2012 R2, 2012 (모두 64비트)
⚙️ 하드웨어 (클라이언트 시스템)
  • CPU: Intel® Core™ i5-8500 이상
  • RAM: 8GB 이상 권장
  • Storage: 650GB 이상 (SATA 7200 RPM Enterprise)
  • Network: Gigabit Ethernet 권장
  • Network: 메인 서버 구축 필요 시 아래 메일로 스펙 문의 필요

  • CPU: Intel® Xeon® Silver 4309Y *2
  • RAM: 64GB 이상
  • 고가용성: Load Balancing 지원
  • 확장성: 최대 100,000대 카메라 지원
🌐 웹 클라이언트 호환성

+
🔑 라이선스 활성화 방법

  • 클러스터 라이선스: 다중 서버 환경 지원
  • Trial License: 30일 평가판 제공

  • 시스템 이전 또는 재설치 시 라이선스 이전
  • 온라인/오프라인 방식 모두 지원
  • 라이선스 사용량 실시간 모니터링
  • 자동 갱신 및 만료 알림 기능

Enterprise 보안 정책

🔒 계정 보안 강화
  • 로그인 5회 실패 시 계정 30분 자동 잠금
  • 최소 8자 이상 복합 패스워드 정책
